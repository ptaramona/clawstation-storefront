# ClawStation — AI Agent Command Center

> Mission control for your AI agent fleet. Monitor, manage, and orchestrate OpenClaw agents from one dashboard.

## What is ClawStation?

ClawStation is a web-based dashboard for managing OpenClaw agent fleets. It provides real-time monitoring, task management, file browsing, workflow visualization, and more — all in one place.

## Key Features

- **Live Dashboard** — Real-time agent status, metrics, and activity charts
- **Digital Office** — Pixel art visualization with live agent status
- **Task Board** — Jira-synced Kanban board
- **Agent Files** — SSH-powered file browser with editor and git history
- **n8n Workflows** — Visual automation monitoring
- **Calendar** — Unified schedule view
- **Mobile Responsive** — Full mobile support

## Requirements

- Node.js 20+
- OpenClaw installed
- Jira Cloud (optional)
- n8n (optional)
